Welcome to CSS

Whenever we want to decorate the page or elements in HTML we use CSS.

Name Purpose
HTML Defines the structure of the web pages
CSS Defines the style of the web pages or elements
JavaScript Defines the behavior of the web pages

CSS Syntax

        selector    {property : value; property : value;}

Selector: Which?? It will select one or more HTML elements in the page
Property: What? What property you want to change
Value How? What is the value you want to specify

Value red

There are 3 ways to specify CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS
