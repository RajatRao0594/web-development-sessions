External CSS

There are 3 ways to specify CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

This is a heading defined with h3 tag. We have included internal css to make it look red.

<link href="css/main.css" rel="stylesheet" type="text/css"> h3 { color: red; } 1. In this case, we specify CSS using a style tag.
2. Normally the style tag is kept in head. But it not mandatory.
3. Specify the color: red to h3 element
4. The style defined will be applicable for all the matching selectors
