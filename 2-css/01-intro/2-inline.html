Inline CSS

There are 3 ways to specify CSS

  1. Inline CSS
  2. Internal CSS
  3. External CSS

This is a heading defined with h3 tag. And inline css to make it look red.

<h3 style="color: red;">This is a heading defined with h3 tag. And inline css to make it look red.</h3> 1. In this case, we specify CSS directly to h3 element.
2. Use style attribute of the element.
3. Specify the color: red
4. The style defined for one element can not be reused for others
